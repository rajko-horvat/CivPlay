With this player you can listen to all of the tunes and sound effects from DOS Civilization (1991).
The EXE needs to be placed in original Civ game directory.
The player supports additional drivers: Pro cards (OPL-3) driver and General MIDI sound driver.

To compile the source code you need Microsoft C Compiler 6.00A.
Compile with: cl /AL PLAYER.C
